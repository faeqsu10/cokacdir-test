# 주차별 리서치 가이드

## 1회차: 온톨로지 기본 개념

### 필수 조사 항목
- 온톨로지 정의와 역사 (철학적 → 정보과학)
- 핵심 구성요소: Class, Property, Individual, Axiom
- 표현 언어 비교: OWL vs RDF vs TypeDB Schema
- 추론(Reasoning)의 개념과 종류
- 실생활 온톨로지 활용 예시

### 발표 준비 포인트
- 비전공자도 이해할 수 있는 비유/예시 준비
- TypeDB와 전통 온톨로지의 차이를 한 장으로 정리

---

## 2회차: 오픈 온톨로지 사례

### 필수 조사 항목
- 민간: Schema.org 구조 분석, DBpedia 활용 사례
- 공공: 한국 공공데이터 온톨로지 현황, 해외 사례 (NIEM, ISA)
- 재난: MOAC, HXL 상세 분석, 국내 재난 온톨로지 현황
- 각 온톨로지의 강점/약점 비교표

### 발표 준비 포인트
- 3개 분야 비교표 (민간 vs 공공 vs 재난)
- 실제 SPARQL 쿼리 예시 시연

---

## 3회차: 재난 데이터 + 설계 방법론

### 필수 조사 항목
- 텍스트 데이터: 재난 뉴스, SNS, 재난 문자, 보고서
- 정형 데이터: 기상 관측, 재난 이력, 인구/지역 통계
- 수집 방법: API 호출, 크롤링, 공공데이터 다운로드
- LLM 활용 온톨로지 설계: 자동 엔티티 추출, 관계 추론

### 발표 준비 포인트
- 실제 데이터 수집 코드 시연
- LLM으로 온톨로지 초안 생성 데모

---

## 4회차: 온톨로지 구현 + 예측 모델

### 필수 조사 항목
- TypeDB 스키마 설계 패턴 (재난 도메인)
- 추론 규칙 작성법
- 예측 모델 후보: 시계열(LSTM, Prophet), 분류(RandomForest, XGBoost)
- 재난 위험도 지표 설계 방법론

### 발표 준비 포인트
- TypeDB 스키마 라이브 코딩
- 간단한 예측 모델 학습/평가 시연

---

## 5회차: LLM 연동 + 통합

### 필수 조사 항목
- GraphRAG: 온톨로지 기반 RAG 아키텍처
- 온톨로지 → 자연어 쿼리 변환 (NL2SPARQL, NL2TypeQL)
- LLM + 예측 모델 결합 패턴
- 재난 시나리오 기반 추론 데모 설계

### 발표 준비 포인트
- 전체 파이프라인 시연 (데이터 → 온톨로지 → 예측 → LLM 추론)
- 5주 성과 종합 발표 자료
